## The Stages

### 1. Clone (Birth)

```bash
git clone matrix-seed my-matrix
cd my-matrix
```

An empty canvas with DNA. The philosophy is there. The structure awaits.

### 2. Personalize (Identity)

Edit The Source. Add your own philosophy. The seed becomes yours.

```
psi/The_Source/00_evolution.md  → Your journey
psi/The_Source/01_self_knowledge.md → Your truths
```

### 3. Grow (Accumulation)

Document sessions. Learn from experience. Let wisdom accumulate.

```
/rrr           → Record what happened
/learn         → Study and synthesize
/oracle        → Align with purpose
```

### 4. Upgrade (Power)

When ready, add capabilities from matrix-reloaded:

```bash
# Add voice
cp -r matrix-reloaded/psi/matrix ./psi/

# Add all commands
cp -r matrix-reloaded/.agent/workflows/* ./.agent/workflows/

# Add personalities
cp -r matrix-reloaded/.claude/agents/* ./.claude/agents/
```

Or simply start with matrix-reloaded for full power immediately.

### 5. Distill (Wisdom)

Extract patterns from your unique journey:

```
Session → Retrospective → Learning → Resonance
```

What did YOUR Matrix discover that others haven't?

### 6. Reunite (Contribution)

Return wisdom to the seed:

```bash
# Fork matrix-seed
# Add your universal insights
# Pull request → The seed evolves
```

Your discovery becomes everyone's foundation.
